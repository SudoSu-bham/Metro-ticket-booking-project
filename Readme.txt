Before running the project make sure that you have some libraries like "PILLOW" , "Sqlite3" , "tkinter" , "pyqrcode" and "pypng"
In order to run this project download all above mentioned libraries

To install libraries
open command prompt as "ADMINISTARATOR" and write the following commands
1. pip3 install pillow
2. pip3 install sqlite3
3. pip3 install tkinter
4. pip3 isntall pyqrcode
5. pip3 install pypng
